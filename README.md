# Purple-Pesterchum
A libpurple/Pidgin plugin for rendering Pesterchum messages

Track project progress: https://github.com/users/garlic-os/projects/7

## Requirements
- libpurple[^1]
- [make](https://command-not-found.com/make)

[^1]: `libpurple-dev` in apt

## Compiling
```bash
$ make
```

## Testing
```bash
$ make testrunner
./testrunner
```
