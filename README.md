# Purple-Pesterchum
A libpurple/Pidgin plugin for rendering Pesterchum messages

## Requirements
- glib v2.76 or higher
- [libpurple-mini](https://github.com/felipec/libpurple-mini)
- [make](https://command-not-found.com/make)

## Compiling
```bash
$ ./configure.sh
$ make  # outputs ./libpesterchum.so
```

## Testing
```bash
$ make testrunner
./testrunner
```
